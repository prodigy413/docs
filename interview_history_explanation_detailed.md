# 面談用：職務経歴説明 詳細版

## 1. 全体概要

私のこれまでの経験は、主に **クラウド環境上での Kubernetes / OpenShift 基盤の構築・運用・保守** を中心としています。  
特に、IBM Cloud と AWS の両方のクラウド環境で、コンテナ基盤、クラウドインフラ、IaC、自動化、監視・ログ管理、障害対応、改善対応まで幅広く担当してきました。

技術領域としては、Kubernetes、OpenShift、AWS EKS、ECS、Fargate、RDS、OpenSearch、Lambda、CloudFront、WAF、VPC 関連リソースなどを利用した経験があります。  
また、Terraform を利用したインフラ構成管理、Python / ShellScript による運用スクリプト作成、GitHub Actions による CI/CD 設定、Datadog、AWS FireLens、Sysdig、LogDNA、Instana などを利用した監視・ログ管理も経験しています。

全体として、私の強みは **クラウド基盤の構築だけでなく、構築後の運用・保守・改善まで一貫して対応できること** です。
